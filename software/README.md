# Software - Hice-Free

Esta carpeta contiene scripts de prueba y control de calidad.

## Contenido planificado

- `test_activation.py` — script para probar la fuerza de activación del botón.
- `test_freezing.py` — medición de temperatura durante la congelación.
- `test_pressure.py` — monitoreo de presión interna de la cápsula.
- `quality_control.py` — pruebas automatizadas de lotes de producción.

## Herramientas sugeridas

- Python con librerías: numpy, matplotlib, pyserial.
- Arduino para control de sensores (temperatura, presión).
- Registro de datos en CSV para análisis.
