# Simulaciones - Hice-Free

Esta carpeta contiene los modelos de simulación para validar el comportamiento del sachet.

## Simulaciones planificadas

| Simulación | Software | Objetivo |
|------------|----------|----------|
| Expansión del gas | CFD (ANSYS Fluent) | Calcular enfriamiento por efecto Joule-Thomson |
| Congelación del gel | Modelo térmico (MATLAB) | Tiempo de congelación vs. temperatura inicial |
| Presión interna | MEF (Abaqus) | Resistencia de la cápsula y el sachet |
| Activación mecánica | Análisis estructural | Fuerza de ruptura de la cápsula |

## Parámetros clave

- CO₂: presión inicial 4–6 bar, volumen 1–2 mL.
- Serpentín de cobre: diámetro 0.5 mm, longitud 15–20 cm.
- Gel: capacidad calorífica similar al agua (4.18 kJ/kg·K).
- Tiempo objetivo de congelación: 10–20 segundos.
