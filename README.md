# Hice-Free: Hielo instantáneo para el canal gastronómico

**Sachet desechable de 4×4×2 cm. Activa con un click. Congela en 10–20 segundos. Formato de venta: packs de 6 o 12 unidades.**

Hice-Free no es un producto médico (aunque puede usarse en emergencias). Está diseñado para **restaurantes, bares, botillerías, eventos y delivery**. Soluciona un problema real: **necesito hielo rápido, no tengo, y no quiero depender de un freezer.**

---

## 1. Problema que resuelve

- Un bar se queda sin hielo en medio del servicio.
- Un delivery de bebidas llega caliente porque la hielera no aguantó.
- Un evento al aire libre no tiene acceso a congelador.
- Una botillería quiere ofrecer hielo instantáneo como valor agregado.

**Hice-Free está siempre listo. No ocupa espacio en el freezer. No se derrite. Se activa en segundos.**

### Formato de venta

| **Pack 12** | 12 sachets | Restaurantes, botillerías, catering |

### Componentes (por sachet)

| Componente | Especificación |
|------------|----------------|
| **Sachet exterior** | Plástico/aluminio multicapa, sellado hermético |
| **Gel térmico** | Agua + polímero superabsorbente + acetato de sodio |
| **Serpentín de cobre** | Espiral de 15–20 cm, espesor 0.5 mm |
| **Cápsula de CO₂** | 1–2 mL a 4–6 bar |
| **Membrana de contención** | Calculada para máxima expansión (sin riesgo de rotura) |
| **Botón de activación** | Click con fuerza de 10–20 N |

---

## 3. Mecanismo de activación (seguro)

1. **Presionar el botón central** con el pulgar (fuerza similar a encender un encendedor).
2. La púa rompe la **cápsula de CO₂**.
3. El gas se expande por el **serpentín de cobre**, enfriándolo drásticamente (efecto Joule-Thomson).
4. El serpentín enfría el **gel térmico**.
5. El gel se congela en **10–20 segundos**, solidificando todo el sachet.

**Seguridad:** La membrana que contiene el gas y el gel está **calculada para soportar la máxima expansión**. No hay riesgo de explosión si se activa correctamente. El diseño impide la activación accidental (botón hundido, no saliente).
