# Hice-Free

[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC_BY--NC_4.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

**Sachet de hielo instantáneo desechable. Activa con un click, congela en segundos.**

Hice-Free es un sachet del tamaño de una pastilla (4×4×2 cm) que contiene un gel térmico y un serpentín de cobre. Al presionar el botón central (estilo cigarro click), se libera gas CO₂ que expande y enfría el serpentín, congelando el gel en 10–20 segundos. El resultado es una pastilla de hielo sólida, lista para usar.

---

## ¿Cómo funciona?

1. **Presiona el botón central** (fuerza 10–20 N).
2. **La púa rompe la cápsula de gas CO₂**.
3. **El gas se expande por el serpentín de cobre**, enfriándolo drásticamente (efecto Joule-Thomson).
4. **El serpentín enfría el gel térmico** que lo rodea.
5. **El gel se congela en 10–20 segundos**, solidificando todo el sachet.

## Especificaciones clave

| Parámetro | Valor |
|-----------|-------|
| Dimensiones | 4×4×2 cm |
| Peso | ~150 g |
| Gel térmico | Agua + polímero superabsorbente + acetato de sodio |
| Serpentín | Cobre 0.5 mm, espiral de 15–20 cm |
| Cápsula de gas | CO₂ a 4–6 bar, 1–2 mL |
| Tiempo de congelación | 10–20 s |
| Activación | Botón tipo click (10–20 N) |
| Uso | Desechable |
| Vida útil | 12–24 meses (sellado) |

## Componentes (tecnologías existentes)

| Componente | Tecnología | Proveedor / Referencia |
|------------|------------|------------------------|
| **Sachet multicapa** | Polietileno/EVOH, 0.2–0.3 mm | Industria de envases flexibles |
| **Gel térmico** | Agua + polímero superabsorbente + nucleador | BASF, Evonik |
| **Serpentín de cobre** | Tubo de cobre 0.5 mm | Industria de refrigeración |
| **Cápsula de gas** | CO₂, 4–6 bar, membrana de aluminio | iSi, Liss, Sodastream |
| **Mecanismo de activación** | Pin perforador + botón flexible | Tecnología de cápsulas de café |
