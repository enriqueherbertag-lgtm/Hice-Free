# Hice-Free: Hielo instantáneo en 10 segundos

[![License](https://img.shields.io/badge/License-Proprietary-red.svg)](LICENSE)
[![EN](https://img.shields.io/badge/English-version-blue.svg)](./README.en.md)

Las bolsas de hielo tradicionales requieren congelación previa, ocupan espacio en el freezer y no están listas cuando las necesitas. Las compresas químicas existentes son incómodas, imprecisas o tardan demasiado.

Hice-Free nace para resolver ese problema.

## Que hace

Hice-Free es un sachet del tamaño de una pastilla (4×4×2 cm) que contiene un gel térmico y un serpentín de cobre. Al presionar el botón central, se libera gas CO₂ que expande y enfría el serpentín, congelando el gel en 10–20 segundos. El resultado es una pastilla de hielo sólida, lista para usar.

## Como funciona

1. Presiona el botón central (fuerza 10–20 N).
2. La púa rompe la cápsula de gas CO₂.
3. El gas se expande por el serpentín de cobre, enfriándolo drásticamente (efecto Joule-Thomson).
4. El serpentín enfría el gel térmico que lo rodea.
5. El gel se congela en 10–20 segundos, solidificando todo el sachet.

## Especificaciones clave

| Parámetro | Valor |
|-----------|-------|
| Dimensiones | 4×4×2 cm |
| Peso | ~150 g |
| Gel térmico | Agua + polímero superabsorbente + acetato de sodio |
| Serpentín | Cobre 0.5 mm, espiral de 15–20 cm |
| Cápsula de gas | CO₂ a 4–6 bar, 1–2 mL |
| Tiempo de congelación | 10–20 s |
| Activación | Botón tipo click (10–20 N) |
| Uso | Desechable |
| Vida útil | 12–24 meses (sellado) |
