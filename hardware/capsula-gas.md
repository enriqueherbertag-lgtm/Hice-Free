# Cápsula de Gas - Hice-Free

## Función

La cápsula contiene el gas refrigerante (CO₂) que al expandirse absorbe calor del serpentín, enfriando el gel.

## Especificaciones

| Parámetro | Valor | Notas |
|-----------|-------|-------|
| Gas | CO₂ | Dióxido de carbono, grado alimentario |
| Volumen | 1–2 mL | Volumen de gas líquido comprimido |
| Presión | 4–6 bar | A 20°C |
| Material de la cápsula | Aluminio o polímero frangible | Con membrana de sellado |
| Diámetro | 8–12 mm | Compatible con sachet de 40×40 mm |
| Altura | 5–10 mm | — |

## Proveedores

| Proveedor | Producto | Notas |
|-----------|----------|-------|
| **iSi** | Cartuchos de CO₂ para soda (8 g) | Muy grande para este formato, se puede fraccionar |
| **Sodastream** | Cilindros de CO₂ (60 L) | Muy grande, no aplica directamente |
| **Liss (China)** | Mini cápsulas de CO₂ de 2 mL | Ideal para este formato, grado alimentario |
| **Nespresso / Dolce Gusto** | Cápsulas de café | Mecanismo de perforación existente, se puede adaptar para CO₂ |

## Mecanismo de perforación

| Componente | Especificación |
|------------|----------------|
| Pin perforador | Acero inoxidable 304, diámetro 0.8–1.0 mm, punta cónica |
| Membrana | Aluminio de 0.05–0.10 mm, con muesca láser para ruptura controlada |
| Fuerza de ruptura | 10–20 N |

## Seguridad

| Consideración | Requisito |
|---------------|-----------|
| Transporte | Cápsula debe cumplir con normativa DOT / UN para gas presurizado |
| Almacenamiento | Proteger de temperaturas > 50°C |
| Manipulación | No debe activarse accidentalmente durante el transporte |

## Referencias

- Norma UN 1013: Dióxido de carbono comprimido
- Certificación DOT 2Q: Cartuchos de gas no recargables
