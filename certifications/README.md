# Certificaciones - Hice-Free

Esta carpeta contiene los planes de certificación y documentos de cumplimiento para el sachet de hielo instantáneo.

## Certificaciones necesarias

| Certificación | Ámbito | Estado |
|---------------|--------|--------|
| FDA / CE | Producto de primeros auxilios | Pendiente |
| ISO 13485 | Dispositivos médicos | Pendiente |
| REACH | Seguridad química (gel, CO₂) | Pendiente |
| RoHS | Materiales eléctricos/electrónicos | Pendiente |

## Pruebas de seguridad requeridas

- **Presión interna**: La cápsula de CO₂ no debe explotar antes de tiempo.
- **Temperatura**: El gel debe alcanzar temperatura de congelación sin fugas.
- **Materiales**: El sachet multicapa debe ser resistente a roturas.
- **Vida útil**: 12–24 meses en condiciones normales.
